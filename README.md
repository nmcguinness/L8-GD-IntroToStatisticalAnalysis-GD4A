# Data Analytics - IntroToR
Introductory R code used in class

## Overview 
This repository contains samples code used to learn core R (and useful packages like magrittr, dplyr, ggplot2) for this module.

## Notes

| Topic | Name | Description | Link |
| :- | :- | :- | :-| 
| 00    | Introduction to User Testing            | What user testing is, how to plan a playtest, what to measure, and how to run sessions ethically and consistently.    | [00-introduction-to-user-testing](/Notes/00-introduction-to-user-testing.md)                       |                              |
| 01    | Foundations of Playtest Data            | Data types (categoric/numeric/text), basic descriptive stats, distributions, and what “good data” looks like.         | [01-foundations-of-playtest-data](/Notes/01-foundations-of-playtest-data.md)                       |                              |
| 02    | Comparing Groups and Testing Hypotheses | Comparing groups, choosing appropriate tests, interpreting results, and avoiding common statistical traps.            | [02-comparing-groups-and-testing-hypotheses](/Notes/02-comparing-groups-and-testing-hypotheses.md) |                              |
| 03    | The Base Pipe Operator                  | Using the built-in pipe to write readable pipelines, including anonymous functions for one-off steps. | [03_pipe](/Notes/03_pipe.md) |
| 04    | Vectors and Data Frames                 | Creating vectors, indexing/subsetting, basic stats, building/inspecting data frames, and handling missing values.     | [04-vector-and-dataframe](/Notes/04-vector-and-dataframe.md)                                       |                              |
| 05    | Data Cleaning with the 4 Verbs          | Cleaning playtest datasets with `select()`, `filter()`, `mutate()`, `summarise()`, and producing report-ready tables. | [05-data-cleaning](/Notes/05-data-cleaning.md)                                                     |                              |
| 06    | Practical Analysis Tools and Reporting  | Practical workflow: turning cleaned data into tables/plots and writing clear commentary for a lab/playtesting report. | [06-practical-analysis-tools-and-reporting](/Notes/06-practical-analysis-tools-and-reporting.md)   |                              |
| 07    | Survey Design                           | Writing playtesting survey questions to maximise insight and support statistical analysis in the lab report.           | [07-survey-design](/Notes/07-survey-design.md) |
